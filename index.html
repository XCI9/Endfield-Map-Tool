使用說明 {{ showInstructions ? '▼' : '◀' }}
  1. 本工具原理為抓取截圖特徵，並在大地圖尋找最相似的位置及縮放大小來進行定位。
  2. 截圖請盡量去除四周的 UI 部分，降低識別錯誤機率
  3. 如果識別一直錯誤，可透過復原來取消當前圖片，請嘗試調整區域(例如去除地圖 UI 、調整框選區域)再次嘗試
  4. 如果識別出的圖片很小，可以嘗試按重新辨識上一圖並指定要用更大的放大倍率
  5. 減少地圖上的顯示標記也可以提高辨識準確率
  6. 放大倍率越小(顯示範圍越大)，辨識成功率越高
